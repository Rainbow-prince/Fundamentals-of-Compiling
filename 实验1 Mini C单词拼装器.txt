                                                                                实验1 Mini C单词拼装器


一、实验内容：

1. 把Mini C源代码中的各类单词（记号）进行拼装分类。
          Mini C语言包含了几种类型的单词（记号）：标识符，关键字，数（包括二进制、十进制、八进制和十六机制的整数、浮点数(含科学记数表示)），字符串、注释、特殊符号（分界符）和运算符号等。
2.输入一个Mini C源文件，输出并列出所有可以拼装的单词（记号）。
3.应该书写完善的软件设计文档。
 

说明：Mini C的单词

1)下面是语言的关键字:
e1se if  int  float double  return void  do while for continue break 

所有的关键字都是保留字,并且必须是小写。

2)下面是专用符号:
_  + ++ +=   - -- -=  -> * *= / /=  % %= <  <= > >= ==  !=  =  ; , ( ) [ ] { }  //行注解   /*  */ 块注解

3)Mini C中，小写和大写字母是有区别的。

4)空格由空白、换行符和制表符组成。空格通常被忽略,除了它必须分开ID、NUM关键字。

5)注释用通常的C语言符号有：//是行注解且不可以超过一行； */ 块注解，可以超过一行。注释不能嵌套。
